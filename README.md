# AIRPORT MANAGEMENT SYSTEM -- GRUP 15
**Alex Lucena, David López, Pol Miguel i Ainoa Gassó**

## Project Overview
This project is a simulation system for airport operations focused on airport structure management, flight handling, and gate allocation at LEBL (Barcelona Airport).
It evolves through multiple versions, progressively adding more realistic functionalities such as flight arrivals, departures, gate assignment logic, visualization tools, and simulation of daily operations.

## Versió 1
Deals with the management of airport information. It includes basic functionalities, focuses on elemental data handling and structure management. The video can be found in the following link: https://youtu.be/yAbg19HH8C8

## Versió 4
Extends the application by taking into account aircraft departures, since departing flights can free gates for incoming aircrafts. We also added several additional features that improve the realism and usability of the system:
  **>> Ecological Footprint of flights:** calculates fuel consumption, CO₂ emissions, and provides comparisons such as equivalent car distance and number of trees needed to absorb the emissions per year.
  **>> Flight Delay Management:** allows the user to add delays to arrivals or departures, simulating real operational conditions. Updated times are reflected in the system.
  **>> Time-based gate status visualization:** by entering a specific time, the system shows the occupancy status of all gates at that moment.
  **>> Interactive terminal visualization:** when displaying the terminal layout, hovering over a gate shows a popup with detailed information. Right-clicking on a gate allows the user to manually assign or remove a flight.
  **>> Simulation mode:** once arrivals and departures are merged (or not), the system simulates hourly changes in gate occupancy throughout the day.
